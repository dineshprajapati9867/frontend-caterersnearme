# Caterers Near Me Frontend

A responsive catering search platform frontend built using React, TypeScript, and Tailwind CSS.

# Features

- View all caterers
- Search caterers by name
- Filter caterers by price
- Caterer details page
- Responsive UI
- Loading spinner
- Skeleton loading

# Tech Stack

- React
- TypeScript
- Tailwind CSS
- React Router DOM
- Vite

# Folder Structure

src
 ┣ components
 ┣ pages
 ┣ interfaces
 ┣ App.tsx
 ┗ main.tsx

# Frontend Setup

## Install dependencies

npm install


## Run frontend

npm run dev

# Environment Variables

Create  .env file:

VITE_API_URL=https://backend-caterersnearme.onrender.com/api/
